# 建築風力設計查詢整合系統 — Claude 工作守則

## 語言規則

所有回覆必須使用**繁體中文**（Traditional Chinese）。無論使用者以何種語言提問，Claude 的所有回應一律以繁體中文書寫，包含說明、程式碼注解建議及錯誤訊息解釋。

## 參考文件

本專案的技術規範依據收錄於 `docs/` 資料夾中的四份 PDF 文件。回答任何關於設計風速、風壓係數、地形係數、陣風反應因子或相關計算方式的問題時，應優先參照以下文件：

- `docs/chapter1.pdf` — 第一章：基本設計風速與地況分類
- `docs/chapter2.pdf` — 第二章：風壓係數與設計風壓計算
- `docs/chapter3.pdf` — 第三章：特殊結構與附加規定
- `docs/pic_tab.pdf`  — 圖表附錄：設計圖表與數值對照表

## 美學設計原則

介面配色採用「典籍書卷」設計語彙，定義於 `shared/theme.css`。新增 UI 元件時應遵循此配色系統，避免引入與古典書卷美學不符的現代高飽和色彩。
